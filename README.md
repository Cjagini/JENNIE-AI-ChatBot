# JENNIE - AI ChatBot Application

A modern, full-stack AI chatbot application powered by Google Gemini API. This application provides a seamless conversational experience with real-time responses.

## Features

**Modern UI** - Beautiful, responsive chat interface  
**AI Powered** - Integrated with Google Gemini 2.5 Flash  
**Conversation History** - Maintains context across messages in sessions  
**Real-time Communication** - Instant message delivery  
**Fully Responsive** - Works on desktop and mobile devices  
**Fast & Efficient** - Optimized performance with FastAPI
